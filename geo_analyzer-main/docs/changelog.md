## Ревизия 2026-06-15 (категории 2GIS)
- Категории POI: добавлен резолвер рубрик 2GIS по названию через Categories API (`/2.0/catalog/rubric/search`), учитывающий регион точки (`/2.0/region/get`). rubric_id в 2GIS зависят от региона, поэтому жёстко зашитые в `config.yaml` ID корректны только для одного города; теперь они резолвятся по `rubric_name` для региона анализируемой точки и кешируются (`cache/rubrics`). Без ключа/сети или при ненайденной рубрике сохраняется исходный ID из конфига — регрессии без API нет.
- Загрузчик POI (`places_loader`) и конкурентный анализ теперь работают по категориям 2GIS через этот резолвер; сравнение точек использует тот же загрузчик, поэтому тоже корректно по категориям и в других городах.
- Добавлены юнит-тесты резолвера (точное совпадение названия, фолбэк, офлайн).

## Ревизия 2026-06-15 (ночь)
- Парковки: добавлен структурный путь загрузки из 2GIS без семантики — запрос `type=parking` и поля `items.capacity`, `items.is_paid`, `items.purpose`, `items.level_count` (см. docs.2gis.com). Реальная вместимость берётся из `capacity` (приоритет над оценкой по площади/типу), платность — из `is_paid`, подземность — из `level_count<0`/`purpose`.
